# Novalife Plugin MyTrucker

Ce plugin Novalife MyTrucker permet de dynamiser le métier de routier en offrant diverses missions à réaliser pour le compte de votre société.

## Installation

1. Téléchargez le fichier `MyTrucker.dll` depuis la page des releases de ce dépôt. 
2. Ajoutez le fichier `MyTrucker.dll` dans le dossier des plugins de votre serveur Novalife.
